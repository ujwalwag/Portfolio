IoT-based UGV Robot for Remote Soil Health Sensing

Description: Developed an end-to-end autonomous system for precision agriculture using an Unmanned Ground Vehicle (UGV). The UGV collects real-time soil, environmental, and GPS data, which is securely transmitted to a local base station for real-time analysis and local data storage.

    Technologies: Raspberry Pi, Arduino Nano, NRF24, Matplotlib, GPS Module, NPK Sensors, DHT11 Sensor, Soil Moisture Sensors, Freenove Software.

    Objective: Design and implement a complete autonomous system for environmental monitoring, leveraging secure device-to-base station communication and local data analysis to support precision farming.

  • Designed and built an autonomous Unmanned Ground Vehicle (UGV) for soil data collection and reconnaissance.
  • Integrated multiple sensors (NPK, temperature, humidity) for real-time data acquisition and a GPS module for location tracking.
  • Configured an NRF24 wireless network to enable reliable data transmission from the UGV to the Base Station.
  • Engineered a real-time data pipeline using Python to process, visualize (Matplotlib), and store sensor data in CSV files.
  • Created a real-time, local dashboard for monitoring UGV status, sensor readings, and location data.
